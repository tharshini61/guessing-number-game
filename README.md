# 🎯 Guessing Number Game (Python Project)

## 📌 Overview
A simple Python game where the computer generates a random number and the user has to guess it within 5 attempts.

The game gives hints like:
- Too high 📈
- Too low 📉
- Correct answer 🎉

---

## 🚀 Features
- Random number generation
- 5 attempts limit
- User input handling
- Conditional logic (if-else)
- Beginner-friendly project

---

## 🛠 Tools Used
- Python 3
- Jupyter Notebook

- for loop
